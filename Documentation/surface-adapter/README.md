# Surface Adapter Documentation Hub

All RenderSurface / SurfaceAdapter migration artifacts now live in this folder. Use it as the single entry point when wiring custom rendering surfaces to MTK.

## Quick Start

1. **Need a TL;DR?** Read `quick-reference.md` for the three adapter patterns and API reminders.
2. **Planning migrations?** Start with `index.md` (file listing + guidance) then dive into `migration.md`.
3. **Decision-making context?** `analysis.md` and `summary.md` cover the architectural rationale and outcomes.
4. **Process record?** `delegation-report.md` documents the Codex delegation that produced these files.

## File Map

| File | Purpose |
| --- | --- |
| `quick-reference.md` | Short guide with adapter patterns and code snippets |
| `index.md` | File manifest and navigation by audience |
| `migration.md` | Step-by-step migration handbook |
| `analysis.md` | Architectural analysis and strategy |
| `summary.md` | Executive summary of the migration |
| `delegation-report.md` | Delegation completion report with deliverables |

Keep any new Surface Adapter docs inside this folder to avoid scattering guidance across the repo.
